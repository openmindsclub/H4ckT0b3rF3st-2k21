#### Description:
 > a quiz app to solve basic math problems

#### Dependencies
 > golang

#### How to run:
```go run quiz.go```

the default number of questions you would be given to solve is 4, to specify the number of questions you wish to answer, run
    ```go run quiz.go -num 2```
